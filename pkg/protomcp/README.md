Package protomcp provides the core library for ProtoMCP protocol
implementations, including JSON-RPC 2.0, MCP (Model Context Protocol), and REST
dispatchers.

## Overview

This package contains the runtime support for code generated by
protoc-gen-protomcp. It implements protocol-specific dispatchers that route
messages to service implementations, along with transport abstractions and
validation utilities.

## Protocol Support

- **JSON-RPC 2.0**: Built on sourcegraph/jsonrpc2 for reliability
- **MCP**: Anthropic's Model Context Protocol for AI assistant integration
- **REST**: HTTP/REST endpoints using google.api.http annotations

## Core Components

- Protocol-specific dispatchers for message routing
- Transport abstractions for HTTP/2 and QUIC
- JSON Schema validation for type safety
- Middleware support using darvaza.org/x/web
- Error handling with protocol-specific mappings

## Usage

```go
import "protomcp.org/protomcp/pkg/protomcp"
```

Generated code imports this package to access protocol implementations,
validation utilities, and transport servers.
